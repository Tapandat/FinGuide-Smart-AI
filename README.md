
# FinGuide AI

AI-Powered Financial Expense Tracking & Analytics Platform

## Features

- User Authentication
- Multi-user Architecture
- Expense Tracking
- Income Tracking
- Budget Planner
- Financial Analytics
- Machine Learning Expense Prediction
- SMTP Email Reports
- PDF Report Generation
- Professional FinTech UI

## Technologies Used

- Python
- Streamlit
- SQLite
- Scikit-learn
- Plotly
- SMTP
- ReportLab

## Run Project

pip install -r requirements.txt

streamlit run app.py
